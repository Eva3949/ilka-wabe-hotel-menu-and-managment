# **App Name**: MenuCentral

## Core Features:

- Menu Display: Display the restaurant's menu in a user-friendly format, categorized for easy browsing.
- Category Management: Allow admins to add, edit, and delete menu categories (e.g., appetizers, entrees, desserts).
- Item Management: Enable admins to create, update, and remove menu items, including details like name, description, price, and availability.
- Real-time Updates: Reflect changes made by the admin instantly on the user-facing menu.
- Suggested Item Tool: AI-powered feature that analyzes popular combinations or trending ingredients to suggest new menu items.

## Style Guidelines:

- Primary color: Warm beige (#F5F5DC) to create a welcoming and comforting feel.
- Background color: Light cream (#FAF9F6), very close to white but warmer.
- Accent color: Muted brown (#A69382) for key actions and highlights.
- Body and headline font: 'Literata', a transitional serif, to bring a literary, slightly vintage and trustworthy feeling.
- Use simple, elegant icons to represent menu categories (e.g., a cup for drinks, a plate for entrees).
- A clean, card-based layout with clear visual hierarchy to present menu items effectively.
- Subtle transitions when updating or filtering the menu to enhance user experience.
